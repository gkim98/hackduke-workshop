CSS

CSS is used to define styling for your html objects. You can create a css file and add it to your page with the link tag (e.g. <link rel="stylesheet" href="index.css">). In your CSS file, you use selectors to choose which elements you're defining styles for. Additional selectors can be added to html objects with the "id" and "class" attributes. Styling is in the format css-attribute: value;.

<element class="class"></element> and <element id="id"></element>

.class-name { ...css }

#id-name { ...css }
